# Incident Playbooks

## Incident Types
- Unexpected execution
- Capital breach
- Data corruption
- Broker outage
- Strategy anomaly

## Universal Response
1. Trigger system kill switch
2. Freeze promotions
3. Capture event logs
4. Assess impact
5. Decide recovery

## Post-Incident
- Root cause analysis
- Event replay
- Strategy review
- Governance sign-off
