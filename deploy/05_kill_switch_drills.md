# Kill Switch Drills

## Kill Switch Types
1. System
2. Strategy
3. Symbol

## Drill Frequency
- Weekly in PAPER
- Monthly in LIVE-SANDBOX
- Quarterly in LIVE

## Drill Steps
1. Trigger kill switch
2. Confirm execution halts
3. Verify event logged
4. Confirm dashboard reflects state
5. Restore system manually
