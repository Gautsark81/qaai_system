# Operator Manual — qaai_system

## Operator Role
The operator supervises, approves, and audits.
The operator never manually trades.

---

## Daily Workflow
1. Review system state
2. Review screening & watchlist
3. Monitor paper trades
4. Review tournament outcomes
5. Approve or reject promotions

---

## Dashboard Pages
- Overview: system health
- Strategies: lifecycle state
- Tournament: SSR & ranking
- Failure Modes: rejections & kills
- Capital: exposure & caps

---

## Emergency Actions
- Activate kill switch
- Suspend strategy
- Freeze live execution

All actions are logged.
