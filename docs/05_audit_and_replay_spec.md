# Audit & Replay Specification

## Event Model
Every action emits a structured event:
- Decision
- Rejection
- Execution
- Promotion
- Adaptation

---

## Replay Guarantee
Given the event log:
- Strategy decisions can be reconstructed
- Risk outcomes can be verified
- Capital exposure can be recalculated

---

## Audit Use Cases
- Regulatory inquiry
- Post-loss investigation
- Strategy forensics
- Performance attribution
